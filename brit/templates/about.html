History

BRIT was significantly improved and extended in the SOILCOM project. The functionality of the core modules was adapted to fit the special needs of modelling the SOILCOM waste stream model. Further, the tool is extended with a completely new Household Waste Collection module and an interface with SimuCF, a simulation tool for aerobic and anaerobic digestion processes. To find out more about the SOILCOM project and the associated partners, please visit:

https://northsearegion.eu/soilcom/

SOILCOM is an Interreg North Sea Region project.

The BRIT prototype was developed as part of the Decision Support Tool of the FLEXIBI project. The development in FLEXIBI included the core modules "Bibliography", "Maps", "Materials" and "Inventories" as well as models and data for the case studies "Greenhouses in Nantes" and "Roadside trees in Hamburg". To find out more about the project and all involved partners, please visit the official project website:

http://www.flexibi-biorefineries.eu

FLEXIBI is a project of the ERA-NET FACCE SURPLUS.
